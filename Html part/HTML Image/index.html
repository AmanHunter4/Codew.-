Have you ever wondered how can you grab the attention of the audience.Well,one way is to add colorful images.In Html,we can iport images too.

Format of an image tag.

img=image.In the src,the source of the image is written and in the alt section the alternate text is written.

When due to some issue, if the image is not displayed.Then, the text written in the alt section is displayed.

Some of the issues are as follows:-

  1. Source not specified.
  2. Invalid source.
  3. Web browser doesn't support the image format.
  4. Loading issues.
