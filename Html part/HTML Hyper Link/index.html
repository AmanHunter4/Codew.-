In html, Hyperlink is denoted by an anchor {a} tag.The link is given in href.

Hyperlink is a word or a pharse or an image which when clicked on redirects us to another webpage of the same website or another website.They are used to link webpages.

  • Some uses of anchor tag are:-
  • Used to create navigation bar.
  • Used to link webpages.
  • By linking many webpages, we can create a website!

In forms, also we can link to webpages.By specifing the url in the action.
