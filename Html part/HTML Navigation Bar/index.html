Hwne we create big websites, it gets difficult to see the content.To solve this purpose, we can use navigation bar.In Html, navigatio bar is denoted as nav.

Navigation bar navigates us through out the website.

Format of the nav.

Inside the nav bar there are some anchor tags.In bigger websites,there are more anchor tags.

We can create nav bars both horizontally and vertically.

By default, nav bars are created horizontally.
