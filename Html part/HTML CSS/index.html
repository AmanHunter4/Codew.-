Css is used to style webpages but before we know about css let's take a look at the ways to link css.

Ways of linking css

  1. Inline css
  2. Internal css
  3. External css

Inline css

  • In inline css, we use style tag to style the tag.
  • Eg:- p style="color:blue" /p

    • Internal css

  • In this type of css in the head tag we use style tag for styling.

    • External css

  • In this type of css, the code is written is an external stylesheet.
  • This type of css can change the entire look of the output.
